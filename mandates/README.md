# Mandates

Structured reference tables for UAE cybersecurity and data-protection mandates.

| File | Description |
|---|---|
| [`master-registry.md`](master-registry.md) | Full table of all 13 mandates with 12 attributes each (regulator, deadlines, penalties, localization, breach-notification obligations, DPO requirements). |
| [`deadline-tracker.md`](deadline-tracker.md) | Cross-sector deadline view — every key date in one table. |

For vertical cuts (mandates relevant to Finance, Healthcare, Real Estate, Retail, Logistics, Education, Professional Services, Telecom), see [`../verticals/`](../verticals/).

For a machine-readable JSON version of the master table, see [`../data/regulations.json`](../data/regulations.json).
